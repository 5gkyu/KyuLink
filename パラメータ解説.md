# KyuLink URL パラメータ仕様

URL にパラメータを付けることで、ページを開いた瞬間に特定の状態を起動できます。  
パラメータは読み込み後に URL から除去されます（履歴に残りません）。

---

## 使用可能なパラメータ一覧

| パラメータ | 型 | 説明 |
|---|---|---|
| `tag` | 文字列 | 指定したタグで絞り込んだ状態で起動 |
| `notag` | `1` | タグなしのブックマークだけを表示 |
| `q` | 文字列 | 検索ワードをセットした状態で起動 |
| `sort` | 文字列 | 並び順を指定して起動 |
| `layout` | `grid` / `list` | レイアウトを指定して起動（localStorage に保存） |
| `theme` | 文字列 | テーマを適用して起動（localStorage に保存） |

複数のパラメータを同時に使用できます。

---

## 詳細

### `?tag=タグ名`
指定タグで絞り込んだ状態でページを開きます。

```
https://example.com/?tag=AI
https://example.com/?tag=Google
```

URLエンコードが必要な文字（日本語など）も使用可能です。

```
https://example.com/?tag=%E9%96%8B%E7%99%BA
```

---

### `?notag=1`
タグが付いていないブックマークだけを表示します。

```
https://example.com/?notag=1
```

---

### `?q=検索ワード`
検索ボックスに文字列をセットし、絞り込んだ状態で起動します。

```
https://example.com/?q=YouTube
https://example.com/?q=%E9%9F%B3%E6%A5%BD
```

---

### `?sort=並び順`
有効な値:

| 値 | 説明 |
|---|---|
| `alpha_en_asc` | 名前順（昇順）※既定 |
| `alpha_en_desc` | 名前順（降順） |
| `date-new` | 新しい順 |
| `date-old` | 古い順 |

```
https://example.com/?sort=date-new
https://example.com/?sort=alpha_en_desc
```

---

### `?layout=レイアウト`
有効な値: `grid` / `list`  
設定は localStorage に保存され、次回以降も維持されます。

```
https://example.com/?layout=grid
https://example.com/?layout=list
```

---

### `?theme=テーマ名`
有効な値:

| 値 | テーマ名 |
|---|---|
| `light` | Light（既定） |
| `dark` | Dark |
| `awake` | Awake |
| `lavender` | Twilight |
| `mint` | Ocean |
| `muse` | Muse |
| `majesty` | Majesty |
| `eof` | EOF |

設定は localStorage に保存され、次回以降も維持されます。

```
https://example.com/?theme=dark
https://example.com/?theme=muse
```

---

## 組み合わせ例

タグ「AI」で絞り込み、新しい順で表示:
```
https://example.com/?tag=AI&sort=date-new
```

検索ワード「Google」でダークテーマ:
```
https://example.com/?q=Google&theme=dark
```

グリッドレイアウト・Museテーマで起動:
```
https://example.com/?layout=grid&theme=muse
```
